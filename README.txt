#Steps to run the Self-driving vehicle assistance system model:

1. Open anaconda prompt in the same folder where main.py is present
2. Enter command to open the GUI:
	python main.py
3. In the GUI select run on video or run on webcam to run the model.
4. Find the results in the result folder.
